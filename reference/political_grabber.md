# gather political site information

this function grabs information on the state, county, and township of
collections

## Usage

``` r
political_grabber(x, y, path)
```

## Arguments

- x:

  an sf data frame of collection points

- y:

  a variable which uniquely identifies each observation

- path:

  a path to the directory holding the BarnebyLivesGeodata
