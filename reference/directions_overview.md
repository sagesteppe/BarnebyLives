# return a quick route summary from googleway::google_directions

This function creates a quick abstract of a google maps route, which
contains the travel time, starting location and the route overview. This
function operates entirely within 'directions_grabber'.

## Usage

``` r
directions_overview(x)
```

## Arguments

- x:

  results from a googleway::google_directions API query

## Examples

``` r
# see 'directions_grabber'
```
