# database_templates

Culled list of templates for different herbarium database management
systems.

## Usage

``` r
database_templates
```

## Format

data frame

## Details

this data.frame is opportunistically gathered from a variety of
herbaria. It shows the formats which a spreadsheet document can be
ordered to for mass uploads.
