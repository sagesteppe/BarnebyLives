# retrieve author results for autonyms

this function is internal to result_grabber, it ensures that an author
is collected from the KEW API for infra-species which have autonyms.

## Usage

``` r
all_authors(genus, epithet, infrarank, infraspecies)
```

## Arguments

- genus:

  derived from result_grabber

- epithet:

  derived from result_grabber

- infrarank:

  derived from result_grabber

- infraspecies:

  derived from result_grabber
