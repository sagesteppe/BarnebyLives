# shipping_examples

A list of a set of some specimens the package author once shipped.

## Usage

``` r
shipping_examples
```

## Format

data frame

## Details

this data.frame is based on some specimens I sent somewhere, or it's
randomly sampled from collection.

``` r
head(shipping_examples)
```
