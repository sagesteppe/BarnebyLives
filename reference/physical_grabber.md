# gather physical characteristics of the site

this function grabs information on the elevation, azimuth, geomorphon,
and geology of the site

## Usage

``` r
physical_grabber(x, path)
```

## Arguments

- x:

  an sf data frame of collection points

- path:

  a path to the directory holding the BarnebyLivesGeodata
