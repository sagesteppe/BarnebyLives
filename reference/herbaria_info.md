# herbaria_info

This is a small data frame of herbaria contact info

## Usage

``` r
herbaria_info
```

## Format

Each is a data frame with columns relevant to letter heads

## Details

this data.frame data set is derived from Index Herbariorum and features
some places many of our collectors send specimens to.
