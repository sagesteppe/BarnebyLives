# Have Google maps help you write directions to a site

This function helps users write directions to a site which is somewhat
near roads.

## Usage

``` r
directions_grabber(x, api_key)
```

## Arguments

- x:

  an sf/tibble/data frame of locations

- api_key:

  a Google developer api key for use with googleway
